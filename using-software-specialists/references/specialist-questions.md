# The Question Each Specialist Asks

When two specialists could both fire, this disambiguates. Each asks a question no one else does — that's their unique value.

| Specialist | The question only they ask |
|-----------|--------------------------|
| [Requirements Analyst](requirements-analyst.md) | "What are we assuming, and what are we explicitly NOT building?" |
| [Project Planner](project-planner.md) | "What's the minimal decomposition where each step is independently verifiable, and is the riskiest work scheduled first?" |
| [Deep Research Agent](deep-research-agent.md) | "Is this source reliable, and do we have enough evidence to act?" |
| [System Architect](system-architect.md) | "What is the simplest architecture that works, and what fails when each component goes down?" |
| [Backend Engineer](backend-engineer.md) | "What if this request is duplicated? What fails when retried?" |
| [Database Designer](database-designer.md) | "What queries will this serve, and what happens at 100M rows?" |
| [Frontend Engineer](frontend-engineer.md) | "What does the user see when this fails? Where does this state live?" |
| [Tester](tester.md) | "What are all the edge cases, and does each test name explain the bug if it fails?" |
| [Quality Engineer](quality-engineer.md) | "Are we testing at the right level? Is this design even testable?" |
| [DevOps Engineer](devops-engineer.md) | "What's the blast radius if this deploy fails, and how fast can we roll back?" |
| [Security Engineer](security-engineer.md) | "What's the insecure default here, and how do we make the safe path the easy path?" |
| [Refactoring Expert](refactoring-expert.md) | "Do we have tests first, and when do we stop?" |
| [Troubleshooter](troubleshooter.md) | "What changed between when it worked and when it broke?" |
| [Performance Engineer](performance-engineer.md) | "Where is time actually spent, and what does the production workload actually look like?" |
| [Technical Writer](technical-writer.md) | "Who will maintain this, and what should we delete first?" |
| [Prompt Engineer](prompt-engineer.md) | "Do we have an eval set, and does this prompt work across models?" |
| [ML Engineer](ml-engineer.md) | "What's the baseline, and what does the held-out test set say about whether this model is actually better?" |
